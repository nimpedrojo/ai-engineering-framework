# Engineering Journal

## Purpose

The Engineering Journal records the significant engineering activities performed during the migration.

It is **not** a diary.

It is **not** a task list.

It is a chronological engineering record that captures:

- important work
- engineering decisions
- AI contribution
- blockers
- lessons learned

The journal provides the evidence required to reconstruct the migration process.

# Journal Rules

Every journal entry must:

- describe one engineering session;
- reference the affected deliverables;
- document AI usage when applicable;
- record important engineering decisions;
- identify blockers;
- define the next action.

Do not record routine work.

Record only work that provides long-term engineering value.
