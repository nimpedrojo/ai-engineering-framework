# AI Migration Harness

# Close Engineering Session

## Purpose

Close the current engineering session.

Synchronize the harness documentation.

Prepare the project for the next working session.

---

## Tasks

Verify that:

- Deliverables have been updated.
- Working State is current.
- Engineering Journal contains today's work.
- Engineering decisions have been recorded.
- Current Slice reflects reality.
- Blockers are documented.
- Next Action is defined.

If anything is missing, report it.

Otherwise, confirm that the session can be closed.

---

## Output

Produce:

- Session checklist
- Missing documentation (if any)
- Confirmation that the harness is synchronized

Do not modify project code.
