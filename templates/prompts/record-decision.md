# AI Migration Harness

# Record Engineering Decision

## Purpose

Create a new engineering decision or update an existing one.

---

## Input

Review:

- Current engineering session
- Engineering Journal
- Working State
- Current Slice

---

## Tasks

Determine whether:

- a new decision is required, or
- an existing decision must be updated.

Populate the decision template.

Assign the next available Decision ID.

Store the decision in:

`docs/ai-harness/decisions/`

---

## Rules

- Record only engineering decisions.
- Ignore implementation details.
- Explain trade-offs.
- Reference affected documents.
- Reference affected slices.

---

## Output

Return the completed decision document.
